# IMPORTANTE: Instalación de Dependencias

Después de hacer pull de este commit, es necesario ejecutar:

```bash
cd backend
npm install
```

Esto instalará las nuevas dependencias:
- `qrcode`: Para generación de códigos QR
- `@types/qrcode`: Tipos de TypeScript para qrcode

La dependencia `bwip-js` ya estaba instalada previamente.
